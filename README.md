## 🚀 Deploy to GitHub Pages (3 steps)

### Step 1 — Create your GitHub repo
1. Go to [github.com](https://github.com) → sign in → click **New repository**
2. Name it: `fuby-tracker`
3. Set to **Public**
4. Click **Create repository**

### Step 2 — Upload your files
Upload ALL files maintaining this exact structure:
```
fuby-tracker/
├── index.html
├── app.js
├── sw.js
├── manifest.json
├── icons/
│   ├── icon-72.png
│   ├── icon-96.png
│   ├── icon-128.png
│   ├── icon-144.png
│   ├── icon-152.png
│   ├── icon-192.png
│   ├── icon-384.png
│   └── icon-512.png
└── .github/
    └── workflows/
        └── deploy.yml
```

**How to upload:**
- Click **Add file** → **Upload files**
- Drag ALL files and the icons folder in
- Click **Commit changes**

### Step 3 — Enable GitHub Pages
1. Go to your repo → **Settings** → **Pages** (left sidebar)
2. Under **Source**, select **GitHub Actions**
3. Wait ~1 minute for the action to run
4. Your app is live at: `https://YOUR-USERNAME.github.io/fuby-tracker`

## 📱 Install on Android
1. Open the URL in **Chrome** on your phone
2. Tap the **Install** banner that appears at the top
3. Or tap the 3-dot menu → **Add to Home Screen**
4. FUBY Tracker is now on your home screen — works fully offline!
